# CSE 340 - Web Backend Development

This repository contains the coursework and projects completed by **Robson Correia** for the **CSE 340: Web Backend Development** course at **Brigham Young University - Idaho (BYU-I)**.

## Course Overview

CSE 340 focuses on server-side web development, covering topics such as:
- Building dynamic web applications using PHP and MySQL.
- Implementing the Model-View-Controller (MVC) design pattern.
- Database design, normalization, and CRUD operations.
- User authentication and session management.
- Input validation and security best practices.

## Course Outcomes

Upon completion of this course, students will be able to:
- Develop web applications that follow frontend standards learned previously.
- Create server-side-rendered applications using modern frameworks and design patterns.
- Design and integrate relational databases for CRUD operations.
- Implement appropriate client-side and server-side validation strategies.
- Articulate the terminology, principles, and best practices of web backend development.
